# 랜딩 페이지 관리자 설정 구현 가이드

1. Manager page : Side Menu -> "landing page" 명칭으로 신규 생성하여 다음과 같은 설정 내용을 구현함.

2. 설정 내용 ('사용되는 모든 경고/확인창을 모던한 모달로 사용 요청)
   1) Logo : 로고 이미지와 텍스트를 사용자가 CRUD 할 수 있는 로직 구현(로고 이미지 사이즈 안내필요함)
   2) Hero Manager : 히어로 섹션의 이미지 및 텍스트를 사용자가 CRUD 할 수 있는 로직 구현
   3) Couple Photo : 3번째 섹션의 couple card를 사용자가 CRUD 기능 로직 구현

3. 텍스트 관련 CRUD 기능이 적용되는 섹션에는 '폰트 종류, 크기, 컬러' 선택할 수 있는 옵션 로직 구현.
